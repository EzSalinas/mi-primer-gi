Comandos para git

  1. Primer comando: git init
  2. Segundo comando: git status (chequea el estado de git en nuestro proyecto)
  3. Tercer comando: git add (trackear todos los archivos)
  4. Cuarto comando: git commit -m "mensaje" (para hacer el paquetito de una version que podemos subir a la nube)
  5. git branch
  6. git remote
  7. git push
